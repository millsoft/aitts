Requirments

  • Active internet connection
  • Google Cloud Text-to-Speech API Key

Voices

Each language has at least one voice. You can select the voice for a language in the settings screen. If no voice was selected, it will use the first WaveNet voice.

Using the TTS

Before you can use this TTS engine, you need to add your Google Cloud Text-to-Speech API Key.

How to get the API Key?

Click here to find out how to get the API Key

How much does it cost?

At the time of writing this, the Google API is free for the first 4 million characters (standard voices) and 1 million characters for the WaveNet voices.

Beside this, be aware that aiTTS engine generates a lot of web traffic which may cause higher mobile costs. It is recommended to use this TTS on good internet to prevent wait time after each sentence.
