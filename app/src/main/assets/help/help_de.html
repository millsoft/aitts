Voraussetzungen

  • Aktive Internetverbindung
  • Google Cloud Text-to-Speech API Key

Stimmen

Jede Sprache hat mindestens eine Stimme. Du kannst die Stimme pro Sprache in den Einstellungen auswählen. Wenn keine Stimme gewählt wurde, wird die erste WaveNet-Stimme verwendet.

Verwendung

Bevor du diese TTS-Engine nutzen kannst, musst du deinen Google Cloud Text-to-Speech API Key hinterlegen.

Klicke hier, um zu sehen, wie du den API Key erhältst.

Was kostet das?

Zum Zeitpunkt der Erstellung ist die Google API für die ersten 4 Millionen Zeichen (Standard-Stimmen) und 1 Million Zeichen (WaveNet-Stimmen) kostenlos.

Beachte außerdem: Die aiTTS-Engine erzeugt relativ viel Datenverkehr. Dadurch können je nach Mobilfunktarif zusätzliche Kosten entstehen. Für gute Reaktionszeiten empfiehlt sich eine stabile Internetverbindung.
